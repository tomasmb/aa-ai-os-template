# Decision entry template

> Used by the assistant when appending to `memory/decisions.md`.

```
### YYYY-MM-DD — <short title>
- **What:** <the decision>
- **Why:** <reasoning in one or two sentences>
- **Who:** <who was involved / whose call was it>
- **Context:** <what project / meeting / conversation>
- **Follow-ups:** <concrete next steps, if any>
- **Promoted to Notion:** <URL if promoted, else "local only">
```
